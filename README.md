Bugreporter
================================

* Author    : justmarkup hallo@justmarkup.com
* Demo      : http://bugspencer.com/bugreporter

What does it detect?
------------

* OS (Version)
* Browser (Version)
* Browserengine
* Browser Dimension
* Browser Language
* Device Dimension
* Color Depth
* Flash (Version)
* Cookie (yes or no)
* LocalStorage (yes or no)
* Adblock (yes or no)
* Zoomlevel
